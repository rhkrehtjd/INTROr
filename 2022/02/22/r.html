기술통계분석

Feb 22, 2022 • 6 min read

  • 정규 분포
    • 도수분포곡선이 평균값을 중앙으로 하여 좌우대칭인 종 모양을 이루고 있다.
    • 가우스 분포라고도 하며, 이 곡선을 가우스 곡선이라고도 한다.
    • 평균과 표준편차에 의해서 그래프의 모양과 위치가 결정된다.
    • 일반적으로 데이터의 분포가 평균을 중심으로 많은 데이터가 모여 있는 특성을 보인다.
    • 중심 극한 정리에 의해서 데이터의 수가 많아질수록 정규 분포를 따른다고 할 수 있다. 하지만 모든 데이터의 분포가 정규분포를 따른다고 가정할 수는 없다. 따라서 정규성을 전제로 하는 분석기법을 적용하기 귀해서는 정규성 검증을 수행해야 한다.
    • 정규 분포의 전체 면적은 1이며, 왜도 = 0, 첨도 = 0 또는 3이다.
    • 정규분포의 확률을 구하기 위해서 평균을 0, 표준편차를 1로 고정하여 표준화한 결과를 표준정규분포라고 한다.
  • 모수와 비모수 검정
    • 모수 검정 : 관측값이 어느 특정한 확룰분포를 따른다고 전제한 후, 그 분포의 모수에 대한 검정을 시행하는 방법
    • 비모수 검정 : 관측값이 어느 특정한 확률분포를 따른다고 전제할 수 없는 경우에 실시하는 검정방법이다.
    • 일반적으로 케이스의 수가 30개 이상이면 중심극한정리에 의해서 정규분포를 따른다는 전제하에 모수 검정을 적용하게 된다. 만약 케이스의 수가 적거나 정확한 정규분포를 검정하기 위해서는 정규성 검증을 수행한다. 이러한 정규성 검정을 통해서 정규분포이면 모수 검정을 수행하고 그렇지 않으면 비모수 검정을 수행한다.
      • 중심 극한 정리 : 표본의 크기가 커질수록 근사적으로 표본의 평균이 모평균과 같고, 분산이 모분산과 같은 정규분포를 취한다는 이론.
      • 정규성 검정방법 : 정규성 검정방법에는 히스토그램과 Q-Q 플롯을 이용한다.
  • 표준정규분포
    • 평균과 분산이 다른 정규분포를 표준화한 것이 표준정규분포이다. 이러한 표준정규분포는 정규분포의 근간이 되는 평군과 표준편차를 0과 1로 고정하는 과정을 의미한다.
    • 용도 : 정규분포의 확률(신뢰구간)을 구할 때 이용한다.
    • 전체 면적 = 1
    • 기술통계량 : 평균 = 0, 표준편차 = 1
    • 표준화 변수 : Z
    • 이 표준화 변수 Z값에 의해서 평균 0을 기준으로 (+-)1$\sigma$의 표준정규분포로 나타낼 수 있다. 또한 이러한 z값의 확률을 구하기 위해서는 표준정규분포 표를 이용하면 된다.
    • z값과 확률 구간
      • 만약, 특정 자료가 정규분포를 따를 때, 평균으로부터 (+-)1$\sigma$만큼의 구간 확률은 P( -1 < Z < 1 ) = 68.3%이다.
      • 이때 어느 학생이 65 ~ 85점 사이의 확률은 표준화 변수로 표현하면 P( -2 < Z < 2 ) = 95.4%가 되고, 60~90점 사이의 확률은 P( -3 < Z < 3 ) = 99.7%이 된다.
      • 즉, 확률 변수 X값의 범위가 커질 수록 확률값은 커지는 당연한 결과를 볼 수 있다.
    • 신뢰수준
      • 사회과학 분야에서 일반적으로 사용되는 신뢰수준은 90%, 95%, 99%이다. 이때의 Z값은 각각 1.65, 1.96, 2.58이다.
    • 신뢰구간
      • 모집단이 정규분포를 이루고 있다면 다음과 같은 모평균의 신뢰구간 추정 식에 의해서 신뢰구간을 계산할 수 있다.
      • 다음은 모평균의 신뢰구간 추정 식이다.
      • 이때 $\alpha$는 유의수준이다. 유의수준이 0.1이면 신뢰수준은 90%가 되며, 유의수준이 0.05이면 신뢰수준은 95%가 되는 것이다.
      • 위 부등식에서 좌측값과 우측값을 이용하여 신뢰구간을 구할 수도 있다.
  • 표본오차
    • 표본에서 계산된 추측값과 모집단의 실제값과의 차이를 의미한다. 즉, 허용 오차를 백분율로 나타낸 것이다. 이러한 표본오차는 신뢰수준이 결정되면 다음과 같은 허용오차 계산식에 의해서 계산할 수 있다.
    • 분산을 아는 경우 p(1-p)를 $\sigma$로 대체할 수 있다.
  • 왜도와 첨도
    • 왜도 : 평균을 중심으로 한 확률분포의 비대칭 정도를 나타내는 지표이다.
      • 즉, 분포의 기울어진 방향과 정도를 나타내는 양을 의미한다. 왜도의 값이 0보다 크면 분포의 오른쪽 방향으로 비대칭 꼬리가 치우치고 왜도가 0보다 작으면 왼쪽 방향으로 비대칭 꼬리가 치우친다. 0에 근접하면 평균을 중심으로 좌우대칭에 가깝다.
      • 비대칭 분포는 평균, 중위수, 최빈수의 관계에 따라서 나눌 수 있다.
      • 음의 비대칭도 : 평균 < 중위수 < 최빈수
      • 양의 비대칭고 : 평균 > 중위수 > 최빈수
      • 평균 = 중위수 = 최빈수, 정규분포가 된다.
      • 분포가 기울어진 방향과 정도를 나타냄
    • 첨도 : 표준정규분포와 비교하여 얼마나 뾰족한가를 측정하는 지표
      • 첨도가 0 또는 3이면 정규분포 곡선을 이루고, 첨도가 0 또는 3보다 크면 정규분포보다 뾰족한 형태로 나타나며 그보다 작으면 정규분포보다 완만한 형태의 곡선을 그린다.
      • 분포도가 얼마나 중심에 집중되어 있는가?
  • 기술통계량 개요
    • 기술통계? 자료를 요약하는 기초적인 통계량으로 데이터 분석 전에 전체적인 데이터 분포의 이해와 통계적 수치를 제공한다.
    • 모집단의 특성을 유추하는 데 이용할 수 있다.
  • 빈도분석
    • 설문 조사 결과에 대한 가장 기초적인 정보를 제공해 주는 분석 방법으로 광범위하게 이용된다.
    • 특히 성별이나 직급을 수치화하는 명목척도나 서열척도 같은 범주형 데이터를 대상으로 비율을 측정하는 데 주로 이용한다.
    • 또는 전체 응답자 중에서 특정 변수값의 범주에 속한 응답자가 차지하는 비율을 알아보고자 할 때 주로 이용한다.
    • 분포의 특성은 빈도수나 비율 등으로 나타낸다.
  • 기술통계분석
    • 빈도분석과 유사하지만 등간척도나 비율척도와 같은 연속적 데이터를 분석할 때 주로 이용한다.
    • 분포의 특성은 표본의 평균값, 중앙값, 최빈값으로 나타내며, 빈도수, 비율, 표준편차, 분산 등으로 표본의 분포를 알 수 있다.
  • 척도별 기술통계량 구하기
    • 전체 데이터 셋의 분포와 특성을 분석하여 결측치와 이상치를 발견하고 데이터 정제를 통해서 척도별로 기술통계량을 구하는 방법에 대해서 알아보자
